Vue全局API

这些引入的模块具体是做什么的，我们顺着下面的代码读下去应该就能知道。

在文件的20行左右，是一个函数 export function initGlobalAPI (Vue: GlobalAPI) {...} 见名知意，这个函数就是负责初始化全局API。 函数参数后的GlobalAPI是Flow的 Interface,用来检查和GlobalAPI相关的类型的。

// config
  const configDef = {}
  configDef.get = () => config
  if (process.env.NODE_ENV !== 'production') {
	configDef.set = () => {
	  warn(
		'Do not replace the Vue.config object, set individual fields instead.'
	  )
	}
  }
  Object.defineProperty(Vue, 'config', configDef)

这里的configDef，注意，是configDef。configDef是类似这前一章分析过的那样，利用了defineProperty()方法和存取器属性（accessor property）。在非生产环境企图修改configDef会有个警告。关于config的具体分析，笔者会在另一篇文章中详述。
